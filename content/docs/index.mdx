## What is OpenClaw Hub?

OpenClaw Hub is an unofficial community resource site for [OpenClaw](https://openclaw.ai) — the open-source autonomous AI assistant. Our goal is to make OpenClaw accessible, safe, and easy to use for everyone.

We are not affiliated with the official OpenClaw project. This site is maintained by the community, for the community.

## What You'll Find Here

- **Skill Directory** — Curated skills with security ratings (A–D), user reviews, and installation guides. Browse at [clawhub.ai](https://clawhub.ai).
- **Security Guides** — How to evaluate skills, set up sandboxed environments, and run OpenClaw safely.
- **Installation Tutorials** — Step-by-step setup for Windows, Mac, and Linux.
- **Channel Integration** — Connect OpenClaw to WhatsApp, Telegram, Discord, and more.
- **Bilingual Support** — All content is available in English and Chinese.

## Getting Started

New to OpenClaw? Start here:

1. Visit the official site at [openclaw.ai](https://openclaw.ai) to download and install.
2. Come back here for our [installation guide](/docs) and troubleshooting tips.
3. Browse the [skill directory](https://clawhub.ai) to extend what your assistant can do.

## A Note on Safety

Because OpenClaw can execute code and access your local filesystem, skill safety matters. Before installing any skill, check its rating on OpenClaw Hub and review what permissions it requests.

When in doubt, run skills in an isolated environment.

## Contributing

OpenClaw Hub is open to contributions. If you want to submit a skill, improve a guide, or translate content, join the discussion on [GitHub](https://github.com/openclaw/openclaw).
